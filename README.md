# Grand Reveal

A premium editorial landing experience for Aetheris Albania, crafted as a modern React application with TanStack Router, Framer Motion, and Tailwind CSS.

## Overview

This project presents a private collection of architectural residences through a polished narrative experience with:

- immersive hero storytelling
- animated collection and villa pages
- a refined inquiry flow
- responsive layout for desktop and mobile

## Tech Stack

- React 19
- TypeScript
- Vite
- TanStack Router + React Query
- Framer Motion
- Tailwind CSS

## Project Structure

- src/routes — file-based route pages for the landing experience
- src/components — shared UI and layout components
- src/data — villa content and metadata
- src/lib — shared utilities and app helpers

## Getting Started

1. Install dependencies
   ```bash
   npm install
   ```

2. Start the development server
   ```bash
   npm run dev
   ```

3. Open the local URL shown in the terminal.

## Available Scripts

- npm run dev — start the local development server
- npm run build — build the production bundle
- npm run preview — preview the built app locally
- npm run lint — run ESLint checks

## Deployment

This project is ready to be deployed to any static hosting platform or server that supports Vite applications.
